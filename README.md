### Sobre o Projeto
O **BookShelf** é uma aplicação mobile focada na organização pessoal de leituras. Substituindo o tradicional papel e caneta, esta app permite ao utilizador monitorizar o seu progresso em tempo real, gerir a sua biblioteca e manter-se motivado para concluir os seus livros.

### Funcionalidades Principais
- **Autenticação Mobile:** Fluxo de Login e Registo optimizado para dispositivos móveis.
- **Navegação Moderna:** Implementação de `BottomNavigationBar` para transições suaves entre abas.
- **Biblioteca Dinâmica:** Uso de `FlatList` (RecyclerView) para renderização de alta performance de grandes coleções.
- **Controlo de Progresso:** Cálculo automático da percentagem de leitura com barra de progresso visual.
- **CRUD Completo:** Gestão total de livros, autores e status de leitura.

### Elementos Técnicos Obrigatórios
- **FlatList:** Listagem optimizada de itens.
- **Bottom Tabs:** Navegação principal por ícones no rodapé.
- **Flexbox Layout:** Interface totalmente adaptável a diferentes tamanhos de ecrã (iPhone/Android).

###  Tech Stack
- **Framework:** React Native (Expo)
- **Navegação:** React Navigation (Stack & Bottom Tabs)
- **Storage:** AsyncStorage (Persistência de sessão)
- **Ícones:** Expo Vector Icons
